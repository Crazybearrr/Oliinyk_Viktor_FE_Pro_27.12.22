Git tags

  • git init- створює в поточному каталозі новий підкаталог з ім'ям .git, що містить усі додаткові файли репозиторію - структуру Git репозиторію.
  • git commit- робить знімок поточних змін проекту.
  • git push- використовується для вивантаження вмісту локального репозиторію в хмарний репозиторій.
  • git status- показує статус робочого дерева.
  • git add- оновлює індекс, призначений для наступного коміту.
